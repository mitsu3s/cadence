# cadence

cadenceは、開発のリズムを見える化して、自己理解とふりかえりの質を高めるサービスです。
GitHubのイベントを収集・可視化して、自分の開発リズム（どれくらいの頻度で、どんなテンポで、どんな状態で開発しているか）を客観的に見える形で提供します。

## Features

- 感覚的にはわかりにくい、自分の開発リズムをデータで把握できます。
- 複数人でのリポジトリであれば、チーム全体の開発リズムも把握でき、振り返りや改善に役立ちます。

## Present condition

- 現在は GitHub の Webhook を利用してイベントを収集しています。
- Cloud Run 上で動作し、Firestore にデータを保存しています。
- APIを通じて、フロントエンドは簡易的なダッシュボードを提供しています。

## Future work

- フロントエンドの充実（より詳細なダッシュボード、グラフ表示など）
- Github Application を利用した認証・連携
- Pub/Sub を利用した非同期処理の導入をはじめとする、クラウド上のアーキテクチャの強化
- オブザーバビリティの向上（ログ、メトリクス、トレースなど）
