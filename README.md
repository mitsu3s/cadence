# cadence

cadence は、開発のリズムを見える化して、自己理解とふりかえりの質を高めるサービスです。
GitHub のイベントを収集・可視化して、自分の開発リズム（どれくらいの頻度で、どんなテンポで、どんな状態で開発しているか）を客観的に見える形で提供します。

## Features

-   感覚的にはわかりにくい、自分の開発リズムをデータで把握できます。
-   複数人でのリポジトリであれば、チーム全体の開発リズムも把握でき、振り返りや改善に役立ちます。

## Present condition

-   Github App (現在は公開前) を通じて、リポジトリのイベントやインストール状況を収集しています。
-   Cloud Run 上で動作し、Firestore にデータを保存しています。
-   API を通じて情報を取得し、ダッシュボードで可視化しています。
-   ダッシュボードでは、Github による認証を必要とします。
